# Consultor Web

Landing page profesional para Alejandro Altamirano, orientada a consultoría técnica, mejora de sistemas empresariales y automatización de datos.

## Stack
- Next.js
- TypeScript
- Tailwind CSS

## Desarrollo local

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```
